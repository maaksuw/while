Ohjeet

Syötteen lukeminen

Ohjelman ensimmäisellä rivillä tulee määritellä syötemuuttujien nimet pilkulla erotettuna seuraavaan tyyliin:
input: x, y;

Syötemuuttujien nimet saa itse päättää, joten voit käyttää eri nimiä kuin tehtävänannossa on käytetty. Oleellista on vain parametrien määrä ja se, että nimet ovat kelvollisia muuttujan nimiä.

Muuttujan nimeäminen

Muuttujan nimen tulee alkaa pienellä kirjaimella ja nimessä saa käyttää englannin kielen aakkosten pieniä ja isoja kirjaimia ja numeroita.

Käskyt

WHILE-ohjelmassa on kolme erilaista käskyä. Esimerkissä a ja b ovat muuttujia.

  • Muuttujaan a asetetaan muuttujan b arvon ja jonkin luonnollisen luvun summa.
    • a = b + 2;
  • Muuttujaan a asetetaan muuttujan b arvon ja jonkin luonnollisen luvun erotus. Mikäli erotuksen arvo on negatiivinen, muuttujaan a asetetaan arvo nolla.
    • a = b - 3;
  • Monista nykyaikaisista ohjelmointikielistä tuttu while-silmukka, jossa toistetaan jotain ohjelman osaa kunnes ehtolause ei ole enää tosi. WHILE-ohjelmissa ehtolause on muotoa muuttuja a on erisuuri kuin nolla. Annettua ohjelman osaa siis toistetaan kunnes silmukkamuuttujan arvo ei ole nolla. 
    • 
while (a != 0) {
    toistettava ohjelman osa
}

Jokainen muuttujan asetus tulee kirjoittaa omalle rivilleen ja päättää puolipisteeseen. Erillistä muuttujan alustuskäskyä ei ole. Kaikki uudet muuttujat alustetaan automaattisesti nollaan. Jos edellisessä esimerkissä b on uusi muuttuja, tulee muuttujan a arvoksi kutsun suorittamisen jälkeen 0 + 2 eli 2. Jos taas edellisessä esimerkissä a on uusi muuttuja, kutsun suorittamisen jälkeen muuttujan a arvo on odotetusti muuttujan b arvon ja luvun 2 summa.

While-silmukassa kaarisulkeet eivät saa olla samalla rivillä, eli seuraava esimerkki ei ole kelvollinen.
while (a != 0) { a = a - 1; }

Vastauksen palauttaminen

Ratkaisun tuottama vastaus luetaan ohjelman suorituksen loputtua ans-nimisestä muuttujasta. Ohjelman antama vastaus pitää siis suorituksen loputtua olla kirjoitettuna ans-muuttujaan. ans-muuttuja on muiden muuttujien tapaan alustettu ensin arvoon nolla ja sitä voi käyttää laskennassa niin kuin mitä tahansa muuta muuttujaa.

Ratkaisun lähettäminen

Kun olet kirjautunut sisään, pääset lähettämään ratkaisun tehtävään. Tehtäväsivulla on tekstikenttä, johon voit liittää tekemäsi ratkaisun. Tekstikenttään voi myös suoraan kirjoittaa vastauksen, mutta voi olla mukavampi koodata ratkaisu ensin tekstieditoriin.

Esimerkkiohjelma

Esimerkkiohjelma saa syötteenä kaksi lukua, jotka on nimetty x ja y. Ohjelma ei tee mitään järkevää, vaan esittelee syntaksin käyttöä käytännössä. 

input: x, y;
a = x + 13;
while (a != 0) {
    y = y + 2;
    b = y - 4;
    while (b != 0) {
        ans = ans + 1;
    }
    a = a - 1;
}
ans = x + 1;
